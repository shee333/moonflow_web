## 模块职责

### 🌟 MoonFlow Core (moonflow_core)

**核心职责**：
- 工作流定义语言 (DSL) 的解析和验证
- 工作流执行引擎
- 任务调度和并发管理
- 40+ 预置组件的实现

**核心文件**：

```
moonflow_core/src/
├── workflow/
│   ├── types.mbt        # 工作流类型定义
│   ├── parser.mbt       # DSL 解析器
│   └── validator.mbt    # 工作流验证器
├── runtime/
│   ├── executor.mbt     # 工作流执行器
│   └── scheduler.mbt    # 任务调度器
├── components/
│   ├── http.mbt         # HTTP 请求组件
│   ├── llm.mbt          # LLM 组件
│   ├── timer.mbt        # 定时器组件
│   ├── database.mbt     # 数据库组件
│   └── ...              # 其他组件
└── demo/
    └── demo.mbt          # 演示示例
```

### 🎨 MoonFlow Web (moonflow_web)

**核心职责**：
- 可视化 DAG 编辑器
- 工作流设计和预览
- 代码编辑器（可选）
- 代码生成（将可视化设计转换为 MoonBit 代码）

**核心文件**：

```
moonflow_web/src/
├── components/
│   ├── DAGEditor.tsx      # DAG 可视化编辑器
│   ├── MoonFlowNode.tsx   # 自定义节点组件
│   ├── ComponentPalette.tsx # 组件选择面板
│   └── types.ts           # 类型定义
├── App.tsx                # 主应用
└── main.tsx               # 入口文件
```

## 数据流

### 1. 可视化设计流程

```
用户操作 → DAG Editor → 内部状态 → 代码生成 → MoonBit 代码
            ↓
        实时预览
```

### 2. 执行流程

```
MoonBit 代码 → Parser → Validator → Executor → Components
                              ↓
                          Scheduler → 并发执行
```

### 3. 部署流程

```
MoonBit 代码 → AOT 编译 → Wasm/Binary → 部署
                         ↓
                    Docker/K8s
```

## 技术选型理由

### 前端技术栈

| 技术 | 选型理由 |
|------|---------|
| React 19 | 成熟稳定，生态丰富，组件化开发 |
| TypeScript | 类型安全，提升开发效率 |
| Vite 5.4 | 快速热更新，开发体验好 |
| ReactFlow | 专为 DAG 可视化设计，功能强大 |
| Monaco Editor | VS Code 同款编辑器，质量保证 |

### 后端技术栈

| 技术 | 选型理由 |
|------|---------|
| MoonBit | 高性能 AOT 编译，突破 GIL 限制 |
| 并发模型 | 原生支持并发，简化调度逻辑 |
| 组件化 | 模块化设计，易于扩展 |

## 扩展性设计

### 添加新组件

1. 在 `moonflow_core/src/components/` 中创建组件文件
2. 实现组件接口
3. 注册到组件注册表
4. Web UI 自动同步显示

### 自定义节点类型

1. 在 `moonflow_web/src/components/types.ts` 中定义类型
2. 创建自定义 React 组件
3. 注册到 ReactFlow

### 代码生成器扩展

1. 实现代码生成接口
2. 支持多种输出格式（MoonBit、JSON、YAML）

## 性能优化

### 前端优化

- ReactFlow 使用虚拟化，只渲染可见节点
- Monaco Editor 按需加载
- 组件懒加载

### 后端优化

- MoonBit AOT 编译，启动快
- 原生并发支持，高吞吐量
- 组件池化复用，减少开销

## 安全性

- 进程级隔离，防止恶意代码
- 能力声明系统，最小权限原则
- 人在回路门禁，关键操作审批
- 可回滚发布，快速恢复

## 监控和调试

### 运行时监控

- 节点执行状态实时显示
- 性能指标收集
- 错误追踪和告警

### 调试工具

- Web UI 实时预览
- 断点调试支持
- 日志查看

## 总结

MoonFlow 采用现代化的架构设计：
- **前后端分离**：灵活的 Web IDE + 高性能后端引擎
- **组件化**：易于扩展和维护
- **可视化**：降低使用门槛
- **高性能**：MoonBit 原生优势
- **安全可靠**：多层防护机制
